# SuplimForge

> พื้นที่รวบรวมโปรเจกต์และผลงาน — ตีขึ้นรูปจากไอเดีย สู่ผลงานจริง

หน้า landing แบบ static (single-file) ธีม **เตาหลอม / Forge** — ตัวอักษรโลหะหลอมเรืองแสง ประกายไฟลอย และแสงเตาหายใจ

## 🔥 เว็บไซต์

เผยแพร่ผ่าน GitHub Pages: **https://suplim2021.github.io/SuplimForge/**

## 📁 โครงสร้าง

- `index.html` — ทั้งหน้าอยู่ในไฟล์เดียว (HTML + CSS ล้วน ไม่ต้อง build, ไม่มี dependency)

## 🛠 พัฒนาต่อ

เปิด `index.html` ด้วยเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไร

## ✨ รายละเอียดงานออกแบบ

- **ฟอนต์:** Roboto Slab (หัวข้อ) + Bai Jamjuree (เนื้อหา/ภาษาไทย) จาก Google Fonts
- **เอฟเฟกต์:** molten gradient text, แสงกวาดผ่าน (shimmer), ประกายไฟลอยขึ้น (embers), แสงเตาหลอมเรืองแสงหายใจ, พื้นผิว grain
- รองรับ `prefers-reduced-motion` และแสดงผลได้ทุกขนาดจอ (responsive)
